# **App Name**: TenantTracker

## Core Features:

- Tenant Management: Add and edit tenant profiles, including names, contact information, and monthly rental rates.
- Status and Rate Management: Enable/disable tenants, set rental rates
- Payment Tracking: Record and track tenant payments with dates, amounts, and payment methods.
- Report Generation: Generate financial reports based on the set time range.
- Delinquency Prediction: Use a generative AI tool to analyze payment patterns and suggest potential overdue accounts based on historical payment behavior.

## Style Guidelines:

- Primary color: A calm blue (#6699CC) to convey stability and trust.
- Background color: Light gray (#F0F0F0), nearly desaturated, for a clean interface.
- Accent color: Muted orange (#D9884B), for emphasis and CTAs.
- Font pairing: 'Inter' (sans-serif) for both body and headlines, due to its clean, modern appearance.
- Use flat, minimalist icons to represent tenant status, payment methods, and report types.
- Employ a grid-based layout for organizing tenant data and payment information efficiently.
- Use subtle transitions and loading animations to provide a smooth user experience.
